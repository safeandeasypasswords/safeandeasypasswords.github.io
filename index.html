Three Word Schema

Preprocessing (to be done only once)

  1. Select two random words with at least 8 letters each, e.g., by opening a dictionary on a random page (e.g., RAPHSODY, EGALITARIAN). The words can be made-up, from other languages etc.
  2. Write down the letters A,B, ..., Z and mark off all letters that appear in your two words (letters not marked: B, C, F, J, K, M, Q, U, V, W, X, Z)
  3. Select a third word that contains many of the letters of the alphabet that aren't marked (i.e., not covered by the first two words) but might contain some that are marked (e.g., MUSICBOX)
  4. Memorize these three words as a fixed sequence (e.g., EGALITARIAN-MUSICBOX-RHAPSODY)
  5. Choose a digit and special character and remember them (e.g., 7, @)

Processing (to be done whenever you need to type a password)

Given a challenge (a website name or prompt based on the website name, e.g., GMAIL), generate a response (password) as follows:

  1. Start at the first letter of the challenge (G).
  2. Find the first occurrence of the letter in your memorized sequence of words and output the next consonant (L in EGALITARIAN). If the letter is Y, wrap back to the first word; if the letter doesn't show up in your sequence of words, skip it.
  3. Repeat on the remaining letters of the challenge till you reach the end (G -> L, M -> S, A -> L, I -> T, L -> T; GMAIL -> LSLTT)
  4. Captialize the first letter, attach your digit and special character; this step is to satisfy password restrictions of most websites (but always use the same rule for yourself and in the same location (beginning, end, after first letter) so you have no trouble remembering). GMAIL ->Lsltt

Another Example

  1. Suppose you are presented with a challenge: AMAZON and you are using the sequence EGALITARIAN-MUSICBOX-RHAPSODY
    2. The first letter of the challenge A is found in the sequence of three words
  2. The output for this letter is the first non vowel following the letter in the sequence, so the output is L
  3. Next we look for M in the sequence and find it in the second word
  4. The next non vowel following M is S, so the output including the second letter of the challenge is LS
  5. The output for the next letter in the sequence, A, is the same as its previous occurence
  6. This means the output including the third letter of the challenge is LSL
  7. The next letter of the challenge, Z, does not appear in our sequence, so we skip this letter
  8. The output does not change and is still LSL
  9. The next letter in the challenge is O which does appear in our sequence
  10. The next non vowel letter in the sequence after O is X, so the output thus far is LSLX
  11. The final letter of our challenge is N which is present in our sequence
  12. the next non vowel letter after N in the sequence is M, so the final output of the challenge AMAZON is LSLXM
